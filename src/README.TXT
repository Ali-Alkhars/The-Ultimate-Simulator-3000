THE SIMULATOR 3000
Ali Alkhars(K20055566) and Anton Sirgue (K21018741)

Description: This project revolved around the simulation of natural habitats. Users can choose a habitat, a climate change scenario, as well as a number of various animals 
and simulate the animals interractions in the selected environment. Simulations can be ended to start a new one and users can interract with their simulation in different ways
through choosing the number of steps they want to run.

!IMPORTANT! Launching the project:
The project's main class is Initializer and not Simulator. To start a new session, user must create an instance of Initializer.
